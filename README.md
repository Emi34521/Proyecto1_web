# Proyecto1_web
Mini Blog is a small front-end demo that lists “posts” backed by the Fake Store API (products). You can browse a paginated grid, open a detail view (with loading and error handling), create and edit entries (demo API), delete from the local list, and filter or search the list client-side by text, category, and max price. The UI uses plain HTML, CSS, and JavaScript—no framework.

Requirements to run

A modern web browser (recent Chrome, Firefox, Safari, or Edge).
Network access so the app can call https://fakestoreapi.com (the list, detail, and form actions depend on it).
Open index.html directly in the browser, or serve the project folder with any static HTTP server (optional; useful if your environment blocks file:// requests or you want consistent behavior).
No Node.js, build step, or package install is required unless you choose to use a local static server of your own.
